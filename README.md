# Tic-Tac-Toe Game 🎮

## 🌟 Features

- Classic tik-tac-toe gameplay.
- Two-player mode
- Real-time Win detection with highlight.
- Game reset functionality.
- Responsive design (works on mobile & desktop).
- Choose between Light and Dark themes based on your preference.
